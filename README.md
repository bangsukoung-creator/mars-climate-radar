# Mars Climate Radar · MCD v6.1

> Real-time atmospheric monitoring of the Martian surface — interactive radar visualization.

## Features

- Original video playback (no color/filter modification)
- Layer toggles: Grid / Data Points / Glitch Effect
- Playback controls: play/pause, speed (0.1x–2x), volume
- Live-simulated climate data overlay (Temperature, Pressure, Dust Opacity, Wind)
- Sparkline charts for data trends
- Responsive layout (desktop + mobile)
- Keyboard accessible (Space = play/pause, Tab navigation)
- Share button (clipboard / Web Share API)

## Tech Stack

- Pure HTML / CSS / JavaScript (no framework, no build step)
- Google Fonts: Share Tech Mono, Orbitron
- Hosted on Vercel (free tier)
